# SkillSwap

SkillSwap is a full-stack freelance marketplace where clients can post tasks and freelancers can browse jobs, submit proposals, and receive payments securely. The platform includes separate dashboards for Clients, Freelancers, and Admins with role-based functionality.

---

## 🌐 Live Website

**Live Link:** https://your-live-link.vercel.app

---

## 🎯 Project Purpose

The goal of SkillSwap is to connect clients with skilled freelancers through a secure and user-friendly platform. Clients can post tasks, review proposals, hire freelancers, and complete payments, while freelancers can discover projects, submit bids, and manage their work efficiently.

---

## ✨ Key Features

* User authentication with Better Auth
* Role-based dashboards (Client, Freelancer, Admin)
* Post, update, delete, and manage tasks
* Browse tasks with search, filtering, and pagination
* Submit and manage freelancer proposals
* Accept or reject proposals
* Stripe payment integration
* Freelancer public profiles
* Dynamic dashboard statistics
* Admin panel for managing users, tasks, payments, and statistics
* Responsive design for mobile, tablet, and desktop
